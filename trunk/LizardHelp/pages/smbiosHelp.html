SMBIOS configuration Help (smbios.plist)

This is the a simple text file with few parameters. Mostly this information is reflected in system profiler information. This file is in an extended XML format called a property list (or plist as the suffix shows).

Boot ROM Version

The boot rom (aka BIOS version). You can take this data from a real mac or from Apple’s site (which I would describe on how to get information on a particular Boot ROM for a particular mac)

Model Name

The Mac Model Family . For instance if you want to emulate as a MacPro , the Family would be “MacPro” (without quotes) , as for a Mac Mini it would be “MacMini” (without quotes)

Model Identifier

The model , for instance you can call yours as “MacPro0,1″ , or “MacPro1,5″ or even follow a real Mac Model. Its up to you

Manufacter

The manufacturer of the Computer. Default its Apple , but then if you want to put your own OEM branding. Feel free to edit it

SMBIOS Version

Its used when fakeSMC is not being used. But since we have fakeSMC , you can easily skip this part. Only use this if you plan to stick with dsmos or if you want SMC Version to be like a RealMac

FSB Speed

Frequency of your FSB (measured in Mhz)

Serial Number

Serial Number of your Hackint0sh

Processor Speed

Your computer clock speed (i.e Intel Pentium D 830 is a 3.00Ghz processor) . Therefore it would be 3000 (Measured in Mhz) (whereby 1Ghz = 1000Mhz)

Memory Speed

Memory clock speed (i.e 533Mhz , 667Mhz). Its measured in Mhz

Bios Date

Date when the firmware was released. This is optional , you could actually skip this.

Memory Type

Use 18 for DDR / 19 for DDR2 / 20 for DDR2 FB-DIMM / 24 for DDR3 – You can get the whole list for SMmemtype by reading Chameleon’s documentation or grabbing the source code

Memory Manufacturer

Manufacturer of your RAM (i.e Kingston , Apple , etc)

Memory Serial

Serial number of your RAM

Memory Part

part number of your RAM
