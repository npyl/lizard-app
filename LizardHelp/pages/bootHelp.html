Boot configuration Help (com.apple.Boot.plist)

Chameleon uses this file as a preferences file - many aspects of Chameleon can be configured with these options.

Graphics Mode

Force booting using a fixed resolution. Hertz setting is optional

Kernel

Path for alternate kernel

Time Out

Boots the default partition (same device as Chameleon, first bootable partition) automatically after waiting num seconds without further user interaction. Pressing the down arrow allows you to select additional options like verbose booting, or allows you to add additional parameters

Theme

Uses the folder Extra/Themes/theme_name as the GUI theme Chameleon uses (default is an embedded "Default" theme)

Nvidia Injection

Automatic device-properties generation for graphics cards. Currently supports only nVidia cards. (defaults to no)

BuiltIn Ethernet

Automatic device-properties generation for ethernet interfaces. (defaults to no)

Disable User Interface

Disables the graphical Chameleon in favor of the all-text version. GUI mode is enabled by default

32bits Compatibility Mode

Launch the kernel in 32bits mode (default is 64). Only avaible with Snow Leopard

Rescan

Enable DVD/CD-ROM rescan mode.

Rescan Prompt

Prompts for CD-ROM rescan mode.

Quiet Boot

No messages or prompts displayed while booting (press F10 while booting to override)

Instant Menu

Force displaying the partition selection menu. (defaults to no)

Wake

Disable wake up after hibernation (enbaled by default).

ForceWake

Force using the sleepimage (disabled by default).

Wait

Prompt for a key press before starting the kernel. (defaults to no)

USBBusFix

Enable the EHCI and UHCI fixes (disabled by default). These make the USB ports "built-in", enable waking by pressing akey or moving the mouse and fix waking the OS to prevent device removal errors.

EHCIacquire

Enable the EHCI fix only (disabled by default)

UHCIreset

Enable the UHCI fix only (disabled by default)

DropSSDT

Skip the SSDT tables while relocating the ACPI tables.

SMBIOSdefaults

Don't use the default values for SMBIOS overriding. If smbios.plist doesn't exist, factory values are kept.

Skip Video BIOS

Skip VideoBIOS Injection to device-properties. (defaults to Yes)

WakeImage

Use an alternate sleepimage file (default path is /private/var/vm/sleepimage).

DSDT

Use an alternate DSDT.aml file (DSDT is automatically loaded from default path of /DSDT.aml or Extra/DSDT.aml.

Default Partition

Sets the default boot partition where X is the drive and Y is the partition.

smbios.plist

Use an alternate smbios.plist file (default path: /smbios.plist /Extra/smbios.plist rd(0,0)/Extra/smbios.plist).

Video Rom

Use an alternate ROM image located @ /file/path (default path: /NVIDIA.ROM)

Legacy Logo

Show the legacy grey apple logo in GUI mode (defaults to no)

Disable Boot Banner

Show boot banner (logo.png in the selected Theme) in GUI mode (defaults to yes)

Devices Properties

Adds support for devices via a hex base64 encoded "EFI strings". Only entry is supported, but more than 1 device can be included within the encoded string.
