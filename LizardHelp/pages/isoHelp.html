Bootable ISO Help

Build a simple and usefull Bootable ISO for your Retail Installation Disc. Compatible Leopard/Snow Leopard

Instructions

Drag files into view, then run the script. You can replace file by dragging one other with same name.
You can't remove a single file, but remove all by pressing "remove all"

If you drag a dsdt.aml or smbios.plist, or NVIDIA.ROM or mach_kernel, the script will automatically link them in com.apple.boot.plist. Make sure files are named as above.

You can specify a custom theme by declaring it into com.apple.Boot.plist, and drag theme's folder with other files. Custom screen resolution is also supported

The script needs three necessary files: cdboot (shipped with Lizard) + com.apple.Boot.plist + Extensions.mkext or Extensions folder.

Process Infos

This box displays informations about the current process. It warm when script failed or succeded, and tell you what to do if something goes wrong. You can unactivate sounds in preferences.

Options

Disable User Interface: can fix "ebios" errors when the chameleon's GUI is enabled

Mount ISO when finished: If the ISO is succefully created, it will be mounted on desktop

Refresh view when succeded: If the ISO is succefully created, remove files from view and temp folders
